# FiguScan Mundial V13 Premium Dark

Versión PWA local-first para subir a Vercel.

## Mejoras principales

- Diseño Premium Dark / lujo deportivo.
- Fondo oscuro, dorado metálico, azul estadio y bordeaux.
- Carga manual con selección/país y nombre de jugador.
- Filtros por estado y por selección.
- Progreso por selección en inicio.
- WhatsApp con mensajes personalizados y negritas.
- Escaneo con recuadro guía, OCR si el navegador permite Tesseract.js y fallback manual.
- Efecto visual de escaneo exitoso con brillo/partículas.
- Conserva el guardado local anterior para no perder figuritas cargadas.

## Subida

Subí todo el contenido de esta carpeta al mismo repositorio conectado a Vercel.
No subas el ZIP: subí los archivos de adentro.

## Importante

La app sigue siendo frontend simple. No requiere Firebase para guardar. Cada celular conserva sus datos en localStorage.
