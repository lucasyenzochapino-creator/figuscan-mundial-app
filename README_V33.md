# FiguScan Mundial V33

Cambios incluidos:
- Base amplia de países/selecciones con banderas y códigos tipo FIFA.
- Búsqueda de país por nombre, código o alias.
- Solo se muestran accesos rápidos de selecciones principales y las ya usadas, para no saturar la pantalla.
- Toast/cartel más chico y temporal, sin re-render fuerte.
- Botón de flash circular dentro del escáner, sin tapar el texto superior.
- Compartir una figurita intenta enviar imagen + texto con Web Share API. Si el celular no lo permite, abre WhatsApp con texto.
- Mejor detección de borde oscuro de la figurita para reducir fondo real.
