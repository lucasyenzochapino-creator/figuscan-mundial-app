# FiguScan Mundial V49

## Cambios

- Carga rápida por lote: pegar muchos números y guardarlos juntos.
- El lote permite seleccionar país/selección para todas las figuritas.
- Si el lote es repetido, permite elegir cantidad de repetidas por figurita.
- Se eliminó `datalist` en campos de país para evitar que el teclado entre y salga en Android/iPhone.
- Se agregaron chips de país rápidos sin reconstruir la pantalla.
- Catálogo de países reforzado: Cabo Verde, Uzbekistán/Ubekistan, Curazao y selecciones británicas corregidas.
- Gales, Escocia e Irlanda del Norte ya no aparecen como “Reino Unido”.
- Se mantiene el guardado de foto V48: sin deformar, sin filtros de color y sobre fondo limpio.

## Cache

- index.html: `app.js?v=49`, `styles.css?v=49`
- service-worker.js: `figuscan-v49-carga-rapida-fifa-teclado`
