# FiguScan Mundial V4

Versión mejorada para subir desde celular.

## Importante
Subí estos archivos al repositorio plano que ya funciona en Vercel.
No subas el ZIP. Subí el contenido de esta carpeta.

## Archivos principales
- index.html
- app.js
- styles.css
- config.js
- manifest.webmanifest
- service-worker.js
- vercel.json
- firestore.rules
- assets/

## Qué mejora
- Inicio rediseñado.
- Tarjetas Tengo / Me faltan / Repetidas clickeables.
- Estados grandes y con colores.
- Eliminar una figurita.
- Seleccionar varias figuritas.
- Eliminar varias juntas.
- Cambiar estado de varias juntas.
- Cámara con recuadro guía.
- Foto recortada a formato figurita.
- Guardado reforzado con respaldo local si Firebase falla.
- WhatsApp más visual con mensaje de marca.
- Opción de compartir imagen resumen si el celular lo permite.

## Reglas de Firestore
Si borrar o guardar falla, pegá el contenido de firestore.rules en Firebase > Firestore > Reglas y publicá.
