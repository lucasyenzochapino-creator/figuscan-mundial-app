# FiguScan Mundial V22

Se reemplaza el escaneo automático poco confiable por captura asistida:

1. Entrar a Escanear.
2. La cámara prende automáticamente.
3. Centrar la figurita completa en el marco vertical.
4. Tocar el marco o “Sacar foto y guardar imagen”.
5. La app recorta, mejora y guarda la imagen junto con la figurita.
6. Completar número, jugador, país y estado.
